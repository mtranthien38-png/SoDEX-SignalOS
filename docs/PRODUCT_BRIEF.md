# SoDEX SignalOS Product Brief

**SoDEX SignalOS** is a Wave 2-ready AI trading operations dashboard built around the buildathon idea:

- **SoSoValue = the brain**: market data, news, ETF flow and research intelligence.
- **SoDEX = the hands**: on-chain market venue, execution workflow, account/trading primitives.
- **SignalOS = the operating layer**: converts intelligence into explainable actions with safety guardrails.

## Core demo flow

1. Open the dashboard and show health checks.
2. Refresh SoDEX market tape and SoSoValue news / ETF flow.
3. Run the Alpha Agent.
4. Explain the score, rationale and suggested paper order.
5. Submit a guarded paper order.
6. Open `/api/demo-proof`, `/api/sodex-docs`, and `/api/sodex-signing-check` to show API seriousness.

## What makes it a stronger Wave 2 submission

The app is not just a UI mockup. It includes server-side API clients, fallback handling, typed-data signing scaffolding, key-safety docs, validation, smoke tests, and a judging checklist. It is designed to demonstrate strong execution, clear workflow logic, useful UX and practical API usage.
